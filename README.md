# PFXTRADER

## Project Documentation

This project is called PFXTRADER. It is designed to help users manage and trade foreign exchange in an efficient manner.

## Website Deployment Process
1. Ensure all code changes are committed to the repository.
2. Push the changes to the main branch of the repository.
3. The website is automatically deployed to the production server using a CI/CD pipeline connected to this repository.

## How to Make Updates
- To make updates, you can create a new branch for your changes. Name the branch appropriately based on the feature or fix you are working on.
- Once your updates are done, push the branch to the repository and create a pull request for review and merging into the main branch.
- Ensure to test the updates locally before pushing to the main branch to avoid breaking changes.
